## Figma MCP

Installed Claude Code Figma plugin.
Connected Claude Code to Figma via MCP (Model Context Protocol) running locally. Used natural language prompts to generate UI directly inside Figma — starting with a basic signup form as a proof of concept.

Early exploration into whether conversational AI can meaningfully accelerate or replace manual component-building in Figma.
